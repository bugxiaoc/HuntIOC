# HuntIOC


该项目用于插件话接口聚合，添加插件后可实现任意平台任意数据查询。

由于涉密，删除了大多数东西，但是平台本身无影响。


## 接口实现

每个插件必须实现Query接口

每个插件必须有一个返回当前结构体函数，用于注册插件。 

每个插件实现后需要在model.go > Init()函数中添加。

其他的自由发挥，HTTP请求可以用tools里的，也可以自己写，其他的公共函数，可以单都在core/tools下新建或添加。
